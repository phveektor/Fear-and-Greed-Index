Fear & Greed Gauge WordPress Plugin

This plugin displays the Crypto Fear & Greed Index using an animated gauge.

Files:
- `fear-greed-gauge.php` - main plugin file
- `includes/class-fg-data.php` - data fetching and caching
- `includes/class-fg-widget.php` - widget class
- `includes/admin-settings.php` - admin settings and tools
- `assets/` - CSS and JS assets (minified versions included)
- `templates/gauge-display.php` - frontend template
- `uninstall.php` - cleanup

Installation:
- Drop into `wp-content/plugins/` and activate. Use Settings > Fear & Greed Gauge to configure.

Development:
- Minified assets included; edit the non-minified files and regenerate minified versions when needed.
